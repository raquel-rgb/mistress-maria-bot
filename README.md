# Mistress Maria Telegram Bot

Bot de Telegram dominante y autoritario para clientes de Mistress Maria.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Create `.env` file:
```
MISTRESS_BOT_TOKEN=your_bot_token_here
```

3. Run the bot:
```bash
python bot.py
```

## Commands

- `/start` - Bienvenida y comandos disponibles
- `/tribute` - Información de tributos
- `/task` - Recibir tarea del día
- `/confess` - Confesar pecados
- `/worship` - Mensaje de adoración
- `/punish` - Recibir castigo
